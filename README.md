# 📖 About

**Matematicando** is a Flutter application created as an academic project to strengthen core mobile development skills.

The app provides a clean and intuitive interface where users can perform basic mathematical operations and calculate their **Body Mass Index (BMI)**.

During the development of this project, the following concepts were explored:

- Flutter widgets
- Material Design
- Navigation between screens
- Form validation
- User input handling
- Mathematical calculations
- Code organization

---

# ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 📏 BMI (Body Mass Index) Calculator

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/mritaamaral/operacoes_matematicas.git
```

Navigate to the project folder

```bash
cd operacoes_matematicas
```

Install dependencies

```bash
flutter pub get
```

Run the application

```bash
flutter run
```
